<h1><img src="https://raw.githubusercontent.com/KimlikDAO/dapp/ana/components/icon.svg" align="center" height="44"> EvmScript</a></h1>

EvmScript is a TypeScript-native toolkit for generating compact EVM programs.
Its first layer is a lowered IR library for typed stack expressions, fragments,
and solver-guided assembly; later, `.tsevm` will add a TS language extension
that lowers into this library.
